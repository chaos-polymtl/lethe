Detailed Description

template<int dim>
class HeatFluxPostprocessor< dim >

Compute the local heat flux vector field within a domain.

The local heat flux is defined as \(\mathbf{q} = -k \nabla T\) where \(k\) is the thermal conductivity of the material and \(\nabla T\) the temperature gradient.

[in]p_idEither fluid ID or solid ID (as initialized in the physical properties). It is used to name the postprocessed variable vector.
