26/**
27 * @brief Solver class for coupled Computational Fluid Dynamics and Discrete Element Method (CFD-DEM) simulations.
28 *
29 * This class implements a two-way coupled CFD-DEM solver that combines the
30 * Volume-averaged Navier-Stokes (VANS) equations for fluid flow with the
31 * soft-sphere Discrete Element Method for particle dynamics. The solver
32 * enables simulation of fluid-particle interactions in systems containing
33 * dense particle suspensions.
34 *
35 * The CFD component uses Galerkin Least Squares (GLS) stabilization for
36 * the VANS equations, while the DEM component handles particle-particle
37 * and particle-wall collisions using the soft-sphere contact model.
38 *
39 * @tparam dim Spatial dimension of the simulation (2 or 3)
40 *
41 * @ingroup solvers
42 */
