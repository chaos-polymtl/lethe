52/**
53 * @class TimeHarmonicMaxwell Auxiliary physics solver for the time harmonic
54 * form of the Maxwell equations.
55 * This class implements an auxiliary physics solver for the ultraweak
56 * formulation of the time-harmonic Maxwell equations using the Discontinuous
57 * Petrov-Galerkin (DPG) method. The solver is designed to work in
58 * three-dimensional spaces and requires the 3 sets of Finite Element spaces
59 * that define the DPG method: the interior trial space that is discontinuous
60 * between elements, the skeleton trace space that lives on the faces of the
61 *mesh elements and connects the interior solutions of adjacent elements, and
62 *the test space that is enriched compared to the trial space and is also
63 * discontinuous between elements.
64 **/
65

500 /**
501 * This helper function projects a 3D tensor onto the tangential plane
502 * defined by the given normal vector. Mathematically, it computes:
503 * \f[
504 * \mathbf{t} = \mathbf{n} \times (\mathbf{v} \times \mathbf{n})
505 * \f]
506 * which removes the normal component of the vector \f$\mathbf{v}\f$,
507 * keeping only the tangential part.
508 *
509 * @tparam dim Spatial dimension.
510 * @param tensor Input vector (field value) to be projected.
511 * @param normal Unit normal vector defining the face orientation.
512 * @return The tangential component of the input vector.
513 *
514 * @note This operation is used to obtain traces in H^{-1/2}(curl) spaces,
515 * where only tangential components are continuous across interfaces. The
516 * inline keyword is enforced to make sure that the tensor operations are
517 * efficient as they are called frequently during assembly.
518 */
519 DEAL_II_ALWAYS_INLINE inline Tensor<1, dim, std::complex<double>>
520 map_H12(const Tensor<1, dim, std::complex<double>> &tensor,
521 const Tensor<1, dim> &normal)
522 {
523 if (dim != 3)
524 {
525 AssertThrow(
526 false,
527 ExcMessage(
528 "The map_H12 function is only implemented for 3D problems."));
529 }
530 auto result = cross_product_3d(normal, cross_product_3d(tensor, normal));
531
532 return result;
533 }
