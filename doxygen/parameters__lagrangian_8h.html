Detailed Description

Parameter structures for Lagrangian (DEM) particle simulations.

This file defines the parameter classes and enumerations used to configure Lagrangian particle simulations in Lethe, including contact force models, particle physical properties, insertion methods, model parameters, floating walls, boundary conditions, grid motion, and post-processing.
