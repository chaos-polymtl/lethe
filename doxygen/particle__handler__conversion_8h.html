void convert_particle_handler ( const parallel::distributed::Triangulation< dim > &  triangulation,

Particles::ParticleHandler< dim > &  ph_out

Converts an input particle handler with a given set of input properties into an output particle handler with a different set of output properties. This is used, notably, to convert DEM simulation particle_handlers into CFD-DEM particle_handlers.

Template Parameters
dimthe number of dimensions
inputthe properties index used for the ph_in
outputthe properties index used for the ph_out
Parameters
[in]triangulationThe triangulation on which the ph_out is stored
[in]ph_inThe input particle_handler that has properties indified by the input.
[out]ph_outThe output partcile_handler that has properties identified by the output.
