Reports the ratio between the minimum cell size and the maximum particle diameter.

This function loops over all locally owned cells in a distributed mesh to find the minimum vertex-to-vertex distance (cell size), then compares it to the maximum particle diameter. It prints out the results using a ConditionalOStream and issues a warning if the ratio is too low.

Template Parameters
dimThe spatial dimension of the triangulation.
Parameters
triangulationA reference to the distributed triangulation (mesh) object.
maximum_particle_diameterThe largest particle diameter in the simulation.
pcoutA ConditionalOStream used for printing output. In parallel simulations, this stream ensures that only one processor (usually the master) prints messages to avoid cluttered output from all MPI ranks.
mpi_communicatorThe MPI communicator used for the parallel simulation. This is required to compute the global minimum cell size across all MPI processes.
