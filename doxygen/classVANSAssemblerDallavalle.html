Detailed Description

template<int dim>
class VANSAssemblerDallavalle< dim >

Class that assembles the drag force using Dallavalle model for the VANS equations where the drag coefficient c_d = pow((0.63 + 4.8 / sqrt(re)), 2) and the momentum exchange coefficient beta =(0.5 * c_d * M_PI * pow(particle_properties[DEM::CFDDEMProperties::PropertiesIndex::dp], 2) / 4) * relative_velocity.norm()
