Detailed Description

template<int dim>
struct Parameters::ConstrainSolidDomain< dim >

Set of parameters constraining a certain portion of a fluid domain to null velocity and pressure fields to mimic a solid subdomain.

Remarks
Pressure DOFs in "solid" cells that are next to "fluid" cells are not constrained.
Template Parameters
dimNumber of dimensions of the problem (2D or 3D).
Note
At the moment, only the temperature field is used to constrain the "solid" domain.
