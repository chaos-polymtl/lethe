Detailed Description

template<int dim>
class LaplaceAssembly< dim >

Class that assembles a Poisson problem for all velocity components and pressure variables. According to the following weak form: d^2 U/dx^2=0 and d^2 P/dx^2=0.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
