Detailed Description

template<int dim>
class CFDDEMSolver< dim >

Solver using the GLS Volume-averaged Navier-Stokes (VANS) solver for fluid and the soft-sphere model of the discrete element method (DEM) to simulate solid-fluid flow with two-way coupling.
