Detailed Description

template<int dim>
class TracerAssemblerSIPG< dim >

Assembles the symmetric interior penalty (SIPG) method (or Nitsche's method) for internal faces. This assembler is only required when solving the Tracer equation using a discontinuous Galerkin discretization.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
