Output the data out to "group_files" vtu files, with a pvtu file and a pvd to store the timing This function outputs the data out to "group_files" vtu file that are orchestrated by a .pvtu file. An additional pvd file takes care of storing the time associated with each .pvtu file.

Parameters
pvd_handlera PVDHandler to store the information about the file name and time associated with it
data_outthe DataOut class to which the data has been attached
foldera string that contains the path where the results are to be saved
file_prefixa string that stores the name of the file without the iteration number and the extension
timethe time associated with the file
iterthe iteration number associated with the file
group_filesthe number of vtu files that will be generated.
mpi_communicatorThe mpi communicator
digitsAn optional parameter that specifies the amount of digit used to store iteration number in the file name
