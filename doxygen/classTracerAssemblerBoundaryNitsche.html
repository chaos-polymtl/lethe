Detailed Description

template<int dim>
class TracerAssemblerBoundaryNitsche< dim >

Assembles Nitsche's method for boundary faces. This assembler is only required when solving the Tracer equation using a discontinuous Galerkin discretization since the boundary conditions have to be weakly imposed.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
