Detailed Description

template<int dim>
class TracerAssemblerCore< dim >

Class that assembles the core of the Tracer equation. This class assembles the weak form of: \(\mathbf{u} \cdot \nabla T - D \nabla^2T = S \) with an SUPG stabilization.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
