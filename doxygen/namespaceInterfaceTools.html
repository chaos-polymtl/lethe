◆ BoundingCoordinates

enum InterfaceTools::BoundingCoordinates : unsigned int

Named indices used to access the bounding coordinates of an isocontour.

This enum defines named indices for the minimum and maximum coordinates stored in IsocontourBoundingValues::bounding_values. The values are ordered by spatial direction: \(x\), \(y\), then \(z\).

Enumerator
x_min 

Minimum x value of the isocontour.

x_max 

Maximum x value of the isocontour.

y_min 

Minimum y value of the isocontour.

y_max 

Maximum y value of the isocontour.

z_min 

Minimum z value of the isocontour.

z_max 

Maximum z value of the isocontour.
