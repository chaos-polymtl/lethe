Detailed Description

template<int dim>
class HeatTransferAssemblerMicrowaveHeatingTimeHarmonicMaxwell< dim >

Class that assembles the time average electromagnetic heating for the heat transfer solver when the time-harmonic Maxwell solver is enabled. The electromagnetic heating is applied as a volumetric source term in the heat equation, and is calculated as \( Q = 1/2(\sigma |E|^2 + \omega \varepsilon_0 \varepsilon_i |E|^2 + \omega \mu_0 \mu_i |H|^2 ) \).

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
Parameters
simulation_controlShared pointer of the SimulationControl object controlling the current simulation.
p_time_harmonic_maxwellStruct that holds all time-harmonic Maxwell model parameters.
