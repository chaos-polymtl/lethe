Detailed Description

template<int dim, typename PropertiesIndex>
class ParticleWallJKRForce< dim, PropertiesIndex >

Calculation of the JKR particle-wall contact force using the information obtained from the fine search and physical properties of particles and walls.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
PropertiesIndexIndex of the properties used within the ParticleHandler.
