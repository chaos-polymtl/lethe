Detailed Description

template<int dim>
class SRFPostprocessor< dim >

Compute the velocity field in the Eulerian frame of reference for simulations carried out in a single rotating frame (SRF).

The velocity in the Eulerian frame is \(\mathbf{u}_\mathrm{E} =\mathbf{u}_\mathrm{L}+ \mathbf{\Omega} \times \mathbf{R}\) where \(\mathbf{\Omega}\) is the velocity of the frame of reference and \(\mathbf{R}\) the position vector.
