Detailed Description

template<int dim, typename PropertiesIndex>
class DEMSolver< dim, PropertiesIndex >

Solver using the soft-sphere model of the discrete element method (DEM) to simulate granular systems.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
PropertiesIndexIndex of the properties used within the ParticleHandler.

◆ smallest_contact_search_criterion

template<int dim, typename PropertiesIndex >
double DEMSolver< dim, PropertiesIndex >::smallest_contact_search_criterion
private

The contact search frequency criterion for the dynamic contact detection method. The value is the smallest value between those two quantities:

  • The diameter of the smallest active cell of a triangulation minus the largest particle radius. \(D_{c,min} - D_{p,max} / 2\)
  • A security factor on the neighboring threshold times the largest particle radius. \(factor * (neighborhood_threshold - 1) * D_{p,max} / 2\)
