Detailed Description

template<int dim>
class VANSAssemblerViscousTorque< dim >

Class that assembles the Viscous dissipative torque due to particle's rotation as defined by Derksen (2004). M_viscous_rotation = pi * pow(dp, 3) * mu * (- omega_p)

The complete model described by Derksen is composed of VANSAssemblerViscousTorque + VANSAssemblerVorticalTorque

dimAn integer that denotes the number of spatial dimensions
