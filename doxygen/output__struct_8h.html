template<int dim, typename VectorType >
using OutputStruct = std::variant<OutputStructPostprocessor<dim, VectorType>, OutputStructSolution<dim, VectorType>, OutputStructCellVector>

Variant handler of the three output structs (OutputStructPostprocessor, OutputStructSolution and OutputStructCellVector). This is used to allow the output of both postprocessors with their respective DoF handlers, and the straightforward cell data struct in a single vector of OutputStruct.
