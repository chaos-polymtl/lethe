Detailed Description

Transforms the signed distance to the phase fraction with a piecewise polynomial function (degree 4).

The phase fraction is defined as:

if \(d < 0 \) \(\phi = 0.5 - 0.5(4d' + 6d'^2 + 4*d'^3 + d'^4)\)

else \(\phi = 0.5 - 0.5(4d' - 6d'^2 + 4*d'^3 - d'^4)\)

with \(d' = d/d_\mathrm{max}\), where \(d\) the signed distance and \(d_\mathrm{max}\) the maximum redistanciation distance. This transformation clamps the phase fraction to 0 or 1 when \(d = \pm d_\mathrm{max}\).
