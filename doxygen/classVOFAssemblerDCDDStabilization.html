Detailed Description

template<int dim>
class VOFAssemblerDCDDStabilization< dim >

Assemble the Discontinuity-Capturing Directional Dissipation (DCDD) stabilization term for the VOF phase fraction.

Note
For more information see Tezduyar, T. E. (2003). Computation of moving boundaries and interfaces and stabilization parameters. International Journal for Numerical Methods in Fluids, 43(5), 555-575. The implementation is based on equations (70) and (79), which are adapted for the VOF solver.
Template Parameters
dimInteger that denotes the number of spatial dimensions.
