Detailed Description

template<int dim>
class CahnHilliardAssemblerCore< dim >

Class that assembles the core of the Cahn-Hilliard equation : \( \frac{d \phi}{dt} + (u \cdot \nabla) \phi - \nabla \cdot (M(\phi)\nabla \eta) = 0 \\ \eta - \frac{\lambda}{\epsilon^2}(\phi^3 - \phi) + \lambda \nabla^2 \phi + \xi \nabla^2 \eta = 0 \).

Template Parameters
dimAn integer that denotes the number of spatial dimensions
