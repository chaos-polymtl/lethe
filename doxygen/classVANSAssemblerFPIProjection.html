Detailed Description

template<int dim>
class VANSAssemblerFPIProjection< dim >

Class that assembles the fluid_particle interactions (FPI) for the VANS equations such as the drag force, when the forces are projected from the particles to the fluid grid.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
