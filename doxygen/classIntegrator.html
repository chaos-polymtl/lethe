Detailed Description

template<int dim, typename PropertiesIndex>
class Integrator< dim, PropertiesIndex >

Base interface for classes that carry out the integration of the velocity and position of particles with inertia.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
PropertiesIndexIndex of the properties used within the ParticleHandler.
