Detailed Description

template<int dim>
class Parameters::Nitsche< dim >

Container class managing multiple Nitsche solid objects.

This class holds a collection of NitscheObject instances and provides parameter declaration and parsing for the Nitsche immersed boundary method section of the parameter file.
