Detailed Description

template<int dim, typename PropertiesIndex>
class ParticlesForceChainsBase< dim, PropertiesIndex >

Base class for the particles force chains contact force models. This class does not implement any of the models, but ensures that an interface without template specialization is available. All of the actual implementation of the models are carried out in the ParticlesForceChains class which is templated by the contact model type.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
PropertiesIndexIndex of the properties used within the ParticleHandler.
