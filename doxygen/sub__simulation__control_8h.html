Detailed Description

Controls smaller simulation blocks within simulations.

Lethe simulations are controlled using the SimulationControl class and its derivative. However, some models contain a sub model with it's own time-stepping. This is the case for the CFD-DEM modules which needs to do a number of DEM iterations per CFD time-step. Whenever this is required, a sub simulation control is used. This file contains classes which are used for sub simulation control. The sub simulation control do not all depend from a base class because they are all very different and don't share a common architecture.
