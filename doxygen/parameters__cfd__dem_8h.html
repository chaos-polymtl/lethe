Detailed Description

Parameter structures for CFD-DEM coupled simulations.

This file defines the parameter classes and enumerations used to configure CFD-DEM coupled simulations in Lethe, including void fraction computation methods, drag models, drag coupling strategies, VANS model selection, and the main CFDDEM parameter structure.
