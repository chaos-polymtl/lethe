The constructor stores the parameters necessary to define the histogram distribution.

Parameters
[in]d_listVector of diameter values.
[in]d_probabilitiesVector of probability values based on the volume fraction with respect to each diameter value.
[in]prn_seedPseudo-random number seed for the diameter generation.
[in]min_cutoffMinimum cutoff diameter.
[in]max_cutoffMaximum cutoff diameter.
[in]distribution_weighting_typeWeighting type of the distribution.
[in]function_type_inDefines whether the input probability values correspond to the PDF or the CDF.
[in]interpolateIndicates whether the diameter sampled values are linearly interpolated between input diameter and probability values.
