Detailed Description

template<int dim>
class HeatTransferAssemblerLaserGaussianHeatFluxVOFInterface< dim >

Class that assembles the laser heating as a surface flux for the heat transfer solver when VOF is enabled. The laser heat flux is applied at the VOF interface (where the phase gradient is non-null).

Template Parameters
dimAn integer that denotes the number of spatial dimensions
Parameters
simulation_controlShared pointer of the SimulationControl object controlling the current simulation
p_laser_parametersShared pointer of the laser parameters
