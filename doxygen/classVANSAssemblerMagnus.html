Detailed Description

template<int dim>
class VANSAssemblerMagnus< dim >

Class that assembles the Lift force using Magnus model.

This implementation follows the formulation in the book "Multiphase Flows with Droplets and Particles" by Crowe et al. (2011).

Template Parameters
dimAn integer that denotes the number of spatial dimensions
