Constructs a cylindrical helix by extruding a disk through a helicoidal path aligned with the z axis when orientation is set to 0;0;0.

Parameters
radius_helixthe radius of the helicoidal path
radius_diskthe radius of the disk that is extruded along the helicoidal path
heightthe total height of the helicoidal path
pitchthe height difference between each helix loop around its axis
positionthe position of the helix base
orientationthe orientation of the helix axis compared to the z axis
