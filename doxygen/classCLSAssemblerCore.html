Detailed Description

template<int dim>
class CLSAssemblerCore< dim >

Class that assembles the core of the CLS solver. According to the following weak form: \(\mathbf{u} \cdot T \cdot \nabla T =0 \) with an SUPG stabilization.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
