16/**
17 * @brief Structure representing a Butcher tableau for a SDIRK method.
18 *
19 * A singly diagonally implicit Runge-Kutta (SDIRK) method is defined by
20 * a Butcher tableau composed of three components:
21 *
22 * - A: a lower triangular matrix of coefficients a_ij, used to compute each
23 * stage.
24 * - b: a vector of weights b_i, used to combine the stage values into the final
25 * solution.
26 * - c: a vector of stage times c_i, indicating at which time fraction (within a
27 * time step) each stage is evaluated.
28 *
29 * These coefficients fully define the behavior of an implicit Runge-Kutta time
30 * integration method and are specific to each SDIRK variant (e.g., SDIRK22,
31 * SDIRK33).
32 */
33struct SDIRKTable
34{
35 FullMatrix<double> A;
36 std::vector<double> b;
37 std::vector<double> c;
38};

40/**
41 * @brief Construct the Butcher tableau (A, b, c) for a given SDIRK method.
42 *
43 * This function returns the Runge-Kutta coefficients for a diagonally implicit
44 * Runge-Kutta (SDIRK) time integration method, such as SDIRK22 (Alexander's
45 * method here) or SDIRK33 (Kennedy-Carpenter method here). These coefficients
46 * define the time-stepping scheme and are used in implicit time integration of
47 * ODEs or PDEs.
48 *
49 * The Butcher tableau is defined as follows:
50 * - A: the matrix of coefficients a_ij
51 * - b: the weights used in the final combination of stages
52 * - c: the time locations of each stage
53 *
54 * @param[in] method The time integration method (e.g.,
55 * TimeSteppingMethod::sdirk22, TimeSteppingMethod::sdirk33) selected from the
56 * Parameters::SimulationControl::TimeSteppingMethod enum.
57 *
58 * @return SDIRKTable A structure containing the Butcher tableau (A, b, c) for the requested method.
59 */

68class SDIRKStageData
69{
70public:
71 /**
72 * @brief Extract the coefficients of a given stage from a SDIRK Butcher tableau.
73 *
74 * This function returns the data needed to compute the solution increment
75 * at a particular stage of a SDIRK (Singly Diagonally Implicit Runge-Kutta)
76 * method. The returned structure contains:
77 * - a_ij: the vector of coefficients from stage j to stage i (for j = 0 to
78 * i),
79 * - c_i: the time coefficient associated with stage i,
80 * - b_i: the weight for combining the result of stage i in the final
81 * solution.
82 *
83 * These coefficients are used to compute each stage value \‍( \boldsymbol{z}_i
84 * \‍) of the Runge-Kutta method according to the formula:
85 *
86 * \[
87 * \boldsymbol{z}_i = \mathcal{F}\left(t_n + c_i \Delta t, \boldsymbol{u}_n +
88 * \Delta t \sum_{j=1}^{i} a_{ij} \boldsymbol{z}_j \right)
89 * \]
90 *
91 * Then, the final solution at the next time step is obtained as:
92 *
93 * \[
94 * \boldsymbol{u}_{n+1} = \boldsymbol{u}_n + \Delta t \sum_{i=1}^{s} b_i
95 * \boldsymbol{z}_i
96 * \]
97 *
98 * where \‍( s \‍) is the total number of stages.
99 *
100 * Pay attention that the stage indices start from 1, but the but indicices
101 * for the matrix and the vectors when coding start from 0.
102 *
103 *
104 * @param[in] table The SDIRK Butcher tableau (structure containing A, b, and
105 * c).
106 * @param[in] stage_i Index of the stage (starting from 1) for which the data
107 * is extracted.
108 * @throws dealii::ExcMessage if the tableau is inconsistent (e.g., wrong
109 * dimensions).
110 */
111 SDIRKStageData(const SDIRKTable &table, const unsigned int stage_i);
112
113 std::vector<double> a_ij;
114 double c_i;
115 double b_i;
116};
