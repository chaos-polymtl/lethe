Detailed Description

template<int dim, int property_start_index, int n_components>
class ParticleFieldQCM< dim, property_start_index, n_components >

Particle fieldy calculator. This class stores the required information for the calculation of the projection of a particle field onto a mesh. This class does not solve any equation, but compartmentalize the necessary information for the projection of the particle field onto a mesh. Multiple instances of this function can be called to establish new particle fields.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
property_start_indexAn integer that indicates at which particle property the field starts
n_componentsThe number of components in the field. This number should be either 1 (a scalar) or dim (a Tensor<1,dim>)
