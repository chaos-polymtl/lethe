Detailed Description

template<typename VectorType>
class PhysicsSolver< VectorType >

Contain all the common elements of a physics solver. It creates the non-linear solver as specified by the user using the parameters file and provides all the necessary elements needed by the solver to solve a physics problem.

Parameters
[in]non_linear_solver_parametersSet of parameters that will be used to construct the non-linear solver.
