35 /**
36  * A class that generates an object to manage a serial triangulation that
37  * represents a solid object. This class can represent solid objects for which
38  * the dimension do not match the spatial dimension. For example, a 2D surface
39  * defined in 3D or a line defined in 2D.
40  *
41  * @tparam dim An integer that denotes the dimensionality of the geometry
42  * @tparam spacedim An integer that denotes the dimension of the space occupied
43  * by the embedded solid
44  *
45  */
