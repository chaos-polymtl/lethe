Detailed Description

template<int dim>
class GLSNavierStokesVOFAssemblerSTF< dim >

Assembles the Surface Tension Force (STF) for the Navier-Stokes equations according to the following equation:

\(\mathbf{F_{CSV}}=\sigma k \nabla \phi \frac{2 \rho}{\rho_0 + \rho_1} \)

Template Parameters
dimAn integer that denotes the number of spatial dimensions
