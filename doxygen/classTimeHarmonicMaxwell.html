Detailed Description

template<int dim>
class TimeHarmonicMaxwell< dim >

solver for the time harmonic form of the Maxwell equations. This class implements an auxiliary physics solver for the ultraweak formulation of the time-harmonic Maxwell equations using the Discontinuous Petrov-Galerkin (DPG) method. The solver is designed to work in three-dimensional spaces and requires the 3 sets of Finite Element spaces that define the DPG method: the interior trial space that is discontinuous between elements, the skeleton trace space that lives on the faces of the mesh elements and connects the interior solutions of adjacent elements, and the test space that is enriched compared to the trial space and is also discontinuous between elements.

◆ map_H12()

template<int dim>
DEAL_II_ALWAYS_INLINE Tensor< 1, dim, std::complex< double > > TimeHarmonicMaxwell< dim >::map_H12 ( const Tensor< 1, dim, std::complex< double > > &  tensor,
const Tensor< 1, dim > &  normal 
)
inlineprivate

This helper function projects a 3D tensor onto the tangential plane defined by the given normal vector. Mathematically, it computes:

\[ \mathbf{t} = \mathbf{n} \times (\mathbf{v} \times \mathbf{n}) \]

which removes the normal component of the vector \(\mathbf{v}\), keeping only the tangential part.

Template Parameters
dimSpatial dimension.
Parameters
tensorInput vector (field value) to be projected.
normalUnit normal vector defining the face orientation.
Returns
The tangential component of the input vector.
Note
This operation is used to obtain traces in H^{-1/2}(curl) spaces, where only tangential components are continuous across interfaces. The inline keyword is enforced to make sure that the tensor operations are efficient as they are called frequently during assembly.
