104 /**
105 * @brief Execute the ray tracing algorithm to find intersection points between
106 * photons and particles.
107 *
108 * @tparam NeighborListType Type of the data structure containing the neighboring
109 * cells.
110 *
111 * @param[in] cell_list Data structure containing the local or ghost
112 * neighboring cell to each local cell in the triangulation.
113 * @param[in,out] photon_intersection_points_map A map containing information
114 * about each intersection point found during the ray tracing. The key of the
115 * map is the particle index of the photon. The value is a tuple containing
116 * the distance between the initial location of the photon and the
117 * intersection point, the intersection point and an iterator to the photon
118 * that needs to be removed.
119 */
