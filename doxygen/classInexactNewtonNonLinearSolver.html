Detailed Description

template<typename VectorType>
class InexactNewtonNonLinearSolver< VectorType >

Non-linear solver for non-linear systems of equations which uses an inexact Newton method with \(\alpha\) relaxation to ensure that the residual is monotonically decreasing. It allows to reuse the matrix and avoid its reassembly for every Newton step.
