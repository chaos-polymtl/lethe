Detailed Description

template<int dim>
class GLSNavierStokesVOFAssemblerNonNewtonianCore< dim >

Assembles the core of the Navier-Stokes equation using a Rheological model to predict non-Newtonian behaviors.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
