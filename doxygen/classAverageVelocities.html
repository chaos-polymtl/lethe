Detailed Description

template<int dim, typename VectorType, typename DofsType>
class AverageVelocities< dim, VectorType, DofsType >

Calculates the time-averaged velocities and pressure \((\langle u \rangle, \langle v \rangle, \langle w \rangle, \langle p \rangle)\) and the independent components of the Reynolds stresses tensor \((\langle u'u' \rangle, \langle v'v' \rangle, \langle w'w' \rangle, \langle u'v' \rangle, \langle v'w' \rangle, \langle w'u' \rangle)\).

Template Parameters
dimAn integer that denotes the dimension of the space in which the flow is solved.
VectorTypeThe vector type used for the solvers (Trilinos (normal or block) or deal.II vectors).
DofsTypeThe type of storage of indices which differs according to vector.

inline

Sanitize the average velocity object after a checkpoint has been read by resetting all of the locally_owned vectors using the locally_relevant vectors. This is necessary because only the locally_relevant_vectors are saved, but the calculation routines expect that the content of the locally_owned vectors match that of the locally_relevant vectors.

Reinitialize the values of sum_velocities_dt_with_ghost_cells vector to 0 and the has_started_averaging flag to false. If the initial time for average velocities parameter is greater than the simulation time, the checkpointed time average velocities are ignored after a restart. This allows users to restart a simulation with a different averaging start time.
