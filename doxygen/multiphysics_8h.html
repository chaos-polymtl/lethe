Detailed Description

Identifiers for multiphysics modules and their solution fields.

This file defines the enumerations used to identify the different physics modules available in Lethe (fluid dynamics, heat transfer, CLS, etc.) and the solution fields they produce. It also provides a utility function to convert a physics name string into its corresponding PhysicsID.

enum PhysicsID : unsigned int

Unique identifiers for each physics module in Lethe.

These identifiers are used to index and refer to the different physics solvers throughout the multiphysics framework.
