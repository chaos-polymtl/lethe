Detailed Description

Abstract class to calculate the tracer reaction prefactor.

This model computes the prefactor \( k = \alpha \, C^{(n-1)} \) such that the full reaction term is assembled as \( r = -k \, C \).

Here, \( \alpha \) represents the reaction constant. This parent class could have children classes that are not based on the power law.
