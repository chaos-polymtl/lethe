Detailed Description

template<int dim>
class TracerAssemblerReaction< dim >

Class that assembles the reaction source term of the Tracer equation. It assembles the term: \( S = - R \= - k C \) Where C is the tracer concentration and k is a reaction rate prefactor that changes depending on the underlying model. For power law of base rate $\alpha$ and order $n$, the model is: \( S = - R \= - (\alpha C^(n-1)) C \).

Template Parameters
dimAn integer that denotes the number of spatial dimensions
