Calculation of the constant rolling resistance torque model using the information obtained from the fine search and physical properties.

The model woks as follows: M_r = - mu_r * R_eff * |F_n| * omega_hat omega_hat = (omega_i - omega_j) / (|omega_i - omega_j|)

Calculation of the viscous rolling resistance torque model using the information obtained from the fine search and physical properties.

The model woks as follows: M_r = - mu_r * R_eff * |F_n| * |V_omega| * omega_hat omega_hat = (omega_i - omega_j) / (|omega_i - omega_j|) V_omega = omega_i × (R_i * n_ij) - omega_j × (R_j * n_ji)
