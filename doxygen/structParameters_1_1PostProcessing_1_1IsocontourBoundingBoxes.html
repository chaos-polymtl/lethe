Detailed Description

Isocontour bounding box parameters. This is used to get the evolution of the bounding values (In 2D: \(x_\mathrm{min}\), \(x_\mathrm{max}\), \(y_\mathrm{min}\), and \(y_\mathrm{max}\); In 3D: \(x_\mathrm{min}\), \(x_\mathrm{max}\), \(y_\mathrm{min}\), \(y_\mathrm{max}\), \(z_\mathrm{min}\), and \(z_\mathrm{max}\)) of one or many specified isocontours. The data of each isocontour is written in a separate file.

Remarks
At the moment, this is only implemented for the variables 'temperature' and 'phase' (CLS phase indicator).
