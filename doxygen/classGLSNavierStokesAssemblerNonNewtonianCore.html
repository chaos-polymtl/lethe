Detailed Description

template<int dim>
class GLSNavierStokesAssemblerNonNewtonianCore< dim >

Class that assembles the core of the Navier-Stokes equation using a Rheological model to predict non-Newtonian behaviors.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
