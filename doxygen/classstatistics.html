Detailed Description

A simple container for storing basic statistical values.

This class holds the minimum, maximum, total, and average values of a dataset. It is initialized with zeros by default. This small class allows us to agglomerate the statistics instead of returning tuples. For example, this class is used to store the kinetic energy of the particles in the DEM model.
