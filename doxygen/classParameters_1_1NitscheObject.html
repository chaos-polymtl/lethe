Detailed Description

template<int dim>
class Parameters::NitscheObject< dim >

Parameters for a single solid object used with the Nitsche immersed boundary method.

This class stores all the parameters needed to define and control an individual Nitsche solid, including its mesh, velocity, temperature, penalization coefficients, particle motion settings, and force/torque calculation options.
