Detailed Description

Filter phase fraction with a hyperbolic tangent function for VOF simulations.

The filtered phase is defined as:

\(\phi' = 0.5 \tanh(\beta(\phi-0.5)) + 0.5\)

where \(\phi'\) is the filtered phase fraction value and \(\beta\) is a model parameter that enables sharper definition when increased.

The filter allows a sharper definition of the interface and clamps the value of the phase fraction between 0 and 1.
