Detailed Description

template<int dim>
class ShearRatePostprocessor< dim >

Compute the shear rate scalar field within a domain.

The shear rate is defined as \(\dot \gamma = \frac{\partial u_i} {\partial x_j} + \frac{\partial u_j}{\partial x_i}\) where \(i\) and \(j\) are dimension component indexes.
