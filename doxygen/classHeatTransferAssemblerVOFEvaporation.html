Detailed Description

template<int dim>
class HeatTransferAssemblerVOFEvaporation< dim >

Class that assembles the evaporation sink for the heat transfer solver at the free surface (air/metal interface) when VOF is enabled.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
Parameters
simulation_controlShared pointer of the SimulationControl object controlling the current simulation.
p_evaporationStruct that holds all evaporation model parameters.
