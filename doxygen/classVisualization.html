Detailed Description

template<int dim, typename PropertiesIndex>
class Visualization< dim, PropertiesIndex >

Building patches of particle properties for visualization. This function is taken from Aspect and dealii and implemented here.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
PropertiesIndexIndex of the properties used within the ParticleHandler.
