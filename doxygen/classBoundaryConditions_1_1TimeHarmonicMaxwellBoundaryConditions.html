Detailed Description

template<int dim>
class BoundaryConditions::TimeHarmonicMaxwellBoundaryConditions< dim >

This class manages the boundary conditions for Time Harmonic Maxwell solver. It introduces the boundary functions and declares the boundary conditions coherently.
