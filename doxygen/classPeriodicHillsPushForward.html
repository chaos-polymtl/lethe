template<int dim, int spacedim>
class PeriodicHillsPushForward< dim, spacedim >

The push_forward & the pull_back classes create the vector_value functions gradient function and because it inherits from Function<spacedim>. (formula is currently Euler and can be changed. See AutoDerivativeFunction documentation)
