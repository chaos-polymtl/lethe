## [Master] - 2026/06/15

### Fixed

- MINOR The `force linear solver continuation` parameter is now honored by the matrix-free Navier-Stokes solver (`FluidDynamicsMatrixFree::solve_system_GMRES`). Previously, a failed GMRES solve always propagated the exception and crashed the simulation, regardless of the parameter value. When the parameter is set to `true`, the failure is now caught and the Newton step continues with the current Newton update. As an additional safeguard, if the failed solve leaves the Newton update with non-finite values, it is reset to zero to avoid propagating `NaN`/`Inf` through the Newton iteration.
