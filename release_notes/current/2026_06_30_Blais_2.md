## [Master] - 2026/06/30

### Changed

- MINOR This PR reduces compilation time by removing unnecessary `#include` directives from widely-included headers. Heavy deal.II and boost headers (e.g. `vector_tools.h`, `dof_renumbering.h`, `data_out.h`, `grid_tools.h`, `grid_generator.h`, the boost serialization archives) were removed from headers that did not use them and, where needed, relocated to the source files that actually require them. This has no impact on simulation results. It just reduces compile time on some machines (especially clusters). [#2048](https://github.com/chaos-polymtl/lethe/pull/2048)
