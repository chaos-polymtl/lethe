## [Master] - 2026/06/17

### Added

- MAJOR This PR adds the possibility to use the AMG preconditioner as the linear solver preconditioner for the Conservative Level Set, Tracer and Cahn-Hilliard auxiliary physics. The preconditioner (`ilu` or `amg`) is selected through the existing `preconditioner` parameter of each physics' linear solver subsection. Assertions were also added so that these physics raise a clear error when an unsupported preconditioner (`lsmg` or `gcmg`, which are restricted to the matrix-free fluid dynamics solver) is requested. A new preconditioner type (`none`), specifying that no preconditioner is used, was added. This preconditioner type is only used for the time harmonic electromagnetism physics which leverages the DPG method. Defensive assertions were added in he time harmonic electromagnetism physics to ensure that the chosen preconditioner is consistent.
