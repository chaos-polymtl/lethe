## [Master] - 2026/06/03

### Added

- MINOR Added a Gaussian filter kernel to the Quadrature-Centered Method (QCM) for void fraction projection, selectable via the new `qcm filter type` parameter (`spherical` or `gaussian`).

### Changed

- MINOR Renamed the QCM parameter `qcm sphere diameter` to `qcm smoothing length` (and the corresponding C++ member `qcm_sphere_diameter` to `qcm_smoothing_length`) to reflect that it parametrises either the averaging-sphere diameter (spherical filter) or twice the Gaussian standard deviation (gaussian filter). The old name `qcm sphere diameter` is kept as a deprecated alias and remains accepted in `.prm` files.
