## [Master] - 2026/06/04

### Added

- MAJOR Adds a non-linear phase change permeability model: Carman-Kozeny (``carman-kozeny phase change``). The model is added in both the single fluid and the CLS assemblers. Both source terms are tested through application tests (phase_change_carman_kozeny and  cls_phase_change_carman_kozeny). The model is set through the new ``permeability model`` parameter that replaces the now deprecated ``Darcy type``  parameter. The model introduces two new parameters: ``Carman-Kozeny permeability area`` and ``Carman-Kozeny division tolerance`` in the ``velocity source`` subsection. The user documentation was updated with the new model and changes to the parameters. [#2013](https://github.com/chaos-polymtl/lethe/pull/2013)
